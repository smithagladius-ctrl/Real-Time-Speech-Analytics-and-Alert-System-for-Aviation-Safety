# **App Name**: AirScribe CVR Management

## Core Features:

- Dashboard: Real-time CVR monitoring with live flight tracking and key metrics.
- Live Monitoring: Visualize 4-channel audio waveforms in real-time using WebSocket streaming. Independent control of Pilot, Co-pilot, Area Mic, and Flight Engineer channels
- Archive & Playback: Access a historical recordings library with advanced search capabilities and detailed audio playback synchronized with flight data.
- Flight Data Correlation: Synchronize CVR audio with flight data recorder (FDR) parameters (altitude, speed, heading) for comprehensive analysis.
- Incident Investigation Tool: AI-powered anomaly detection for detecting potential incidents. This feature incorporates the AI tool to match audio events with flight events, such as unusual noises during ascent or descent.
- Reports & Analytics: Generate compliance reports and analytics dashboards for aviation regulatory standards.
- Alerts & Notifications: Manage alerts with a centralized alert management system to improve aviation safety.

## Style Guidelines:

- Primary color: Black Sea (#0A2A3D), evoking the professional aviation atmosphere requested by the user.
- Background color: Dark Blue (#121E2B), close in hue to the primary but significantly desaturated, to complement the requested dark mode look.
- Accent color: Turquoise (#40E0D0), an analogous hue to provide a pop of high-contrast brightness that separates this professional look from 'flat' dark mode interfaces.
- Body and headline font: 'Inter', a grotesque-style sans-serif providing a modern and neutral aesthetic. 
- Code font: 'Source Code Pro' for displaying any code snippets, like JSON responses.
- Use aviation-themed icons, and a separate icon style for alerts and status indicators
- Subtle animations for flight data updates and system alerts, enhancing user awareness.
